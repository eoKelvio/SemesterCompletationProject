HTML/CSS

Em geral nos fizemos diversos projetos relacionados a programação em HTML/CSS e Javascript. Exploramos dês de comandos basicos como <h1>, <p>, <bolder>, <stronger>, que são comandos que em geral servem para alterar textos. Aprendemos também a criar links com o comando <a>, que podem linkar outras páginas dentro do mesmo projeto, ou a sites exteriores. Vimos como colocar imagens usando o comando <img> onde deve ser colocado o link para a imagem dentro do href. E um dos comandos mais importantes foi o <input> que pode criar caixas de texto.

HTML5

Vimos também HTML Semântico, que facilitou muito fazer sites organizados. Aprendemos <container>, <section>, <header>, <nav>, <main>, <footer>, que são todos comandos para organizar a estrutura do site. Vimos também alguns comandos no css para organizar, por exemplo justify-content e align-itens.

Javascript

Em Javascript aprendemos comandos como alert que coloca um alerta na tela. OnMouseClick, OnMouse, que são comandos que acontecerão ao você fazer algo com o mouse. If, else, case que são comandos que funcionam através de condições. While, do while, for que são comandos que criam laços de repetição.

TransformCode

Esse foi o projeto final da matéria. Um grupo de alunos criou um HTML/CSS para os exercícios de Js que haviamos feito, e então teriamos que criar uma página que houvessem dois campos, um você colocaria o HTML e o outro o Javascrip e então o site automáticamente colocária tudo que você fez com o CSS já pronto.
